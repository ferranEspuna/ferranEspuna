# Mandelbrot & Julia (z^2 + c)

Interactive GLSL shader visualizing the Mandelbrot set (parameter plane) and Julia sets (dynamical planes) for quadratic polynomials.

View the interactive visualizer on the website:
[https://ferran.info/complex_fractals/z2_plus_c/](https://ferran.info/complex_fractals/z2_plus_c/)
